阶乘(Factorial)函数
===

Factorial 称之为阶乘，维基百科是这样描述的“一个正整数的阶乘是所有小于及等于该数的正整数的积，并且有 `0` 的阶乘为 `1`。自然数 `n` 的阶乘写作 `n!`。”

阶乘函数是递归(Haskell)函数典型示例。

阶乘定义：
* `0` 的阶乘是 `1`
* 任何数的阶乘都是此数乘以比此数小一的数的阶乘

2. `while` 循环实现：
```js
function factorial(num) {
  var result = num;

  if (num < 0) {
    return -1;
  } else if (num === 0 || num === 1) {
    return 1;
  } else {
    while (num > 1) {
      num--;
      result *= num;
    }
  }

  return result;
}
```

3. `for` 循环遍历：
```js
function factorial(num) {
  if (num < 0) {
    return -1;
  } else if (num === 0 || num === 1) {
    return 1;
  } else {
    for (var i = num - 1; i >= 1; i--) {
      num *= i;
    }
  }

  return num;
}
```
